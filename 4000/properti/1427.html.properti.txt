<title>China Berpeluang Lanjutkan Proyek Pembangunan Kereta Api di Filipina</title>
<first>Peluang China untuk melanjutkan pembangunan tiga mega proyek perkeretaapian di Filipina semakin besar.Departemen Perhubungan Dephan Filipina menyampaikan Menteri Transportasi Filipina, Jaime J.Bautista dan Duta Besar China Huang Xilian sepakat untuk memulai kembali negosiasi untuk proyek proyek transportasi besar di negara berjulukan Pearl of the Orient Seas</first>
<second>.Pertemuan resmi tersebut telah berlangsung di Kedutaan Besar China di Kota Makati, pada Kamis 11/8/2022.“Kedua pejabat tersebut membahas dimulainya kembali pembicaraan untuk proyek proyek kereta api besar yang didanai China seperti Proyek Jarak Jauh Selatan PNR Kereta Api Komuter Utara Selatan, Kereta Api Subic Clark, dan Kereta Api Mindanao Tagum Davao Digos,” ujar Departemen Perhubungan Filipina seperti dikutip dari Business World.</second>
<third>Sementara itu dalam pernyataan terpisah, Duta Besar China Huang Xilian berharap bahwa kerja sama China Filipina di bidang infrastruktur dan perkeretaapian akan mencapai hasil yang lebih nyata dan membawa lebih banyak manfaat bagi Filipina di awal.“China sudah mengabdikan diri untuk mempromosikan konektivitas negara negara Asia Tenggara dan memperkenalkan teknologi mutakhir konstruksi kereta api dan manufaktur rolling stock,” kata Huang.Presiden Ferdinand R</third>
<fourth>.Marcos, Jr.bulan lalu mengarahkan Departemen Pertahananan untuk kembali ke meja perundingan demi mengamankan perjanjian pinjaman untuk tiga proyek perkeretaapian.</fourth>
<fifth>Padahal sebelumnya proses negosiasi mandek di tengah jalan karena pemerintah China dianggap tidak tanggap.Dikatakan, pada kesempatan tersebut Jaime dan Huang juga membicarakan proyek kerjasama maritim China Filipina, khususnya komunikasi hotline dan kerjasama urusan hukum antara Penjaga Pantai Filipina dan Penjaga Pantai China.Ada juga diskusi tentang kemungkinan kerjasama dalam keselamatan lalu lintas maritim, keselamatan feri, serta peningkatan kapasitas untuk tata kelola maritim, pengawasan keselamatan, dan inspeksi keselamatan kapal.</fifth>
