<title>3 Cara Ubah PDF ke Excel: Pakai Word hingga Online</title>
<first>Excel merupakan salah satu aplikasi yang cukup nyaman digunakan untuk melakukan pekerjaan yang berhubungan dengan tabel.Jika Anda pernah mengalami memiliki dokumen berisi tabel namun dokumen dalam bentuk PDF, Anda mungkin bingung mengenai bagaimana cara ubah dokumen PDF ke Excel agar bisa diedit.</first>
<second>Anda tidak perlu cemas.Ada sejumlah cara untuk ubah pdf ke excel sehingga nantinya dokumen PDF bisa terbuka dengan mudah di Excel.</second>
<third>Berikut ini cara untuk mengubah PDF ke Excel dengan sangat praktis, tanpa perlu instal aplikasi tambahan:1.Ubah PDF ke Excel menggunakan WordDikutip dari BusinessInsider , Anda bisa dengan mudah mengubah file PDF menjadi Excel dengan cara membukanya dulu menggunakan Microsoft Word.</third>
<fourth>Adapun cara ubah PDF ke Excel memakai Word yakni sebagai berikut:Klik kanan pada file PDF yang Anda milikiSelanjutnya klik “Open With” dan pilih “Word” jika Anda tidak menemukan pilihan tersebut maka klik “Choose another app” lalu pilih “More App”Setelah pilihan Word dipilih klik “OK”Word akan terbuka otomatis, namun Anda akan melihat kota dialog yang meminta izin untuk mengonversi PDF ke Word, klik “Oke” dan tunggu beberapa menit hingga dokumen terbukaSelanjutnya, di dokumen yang sudah terbuka pada Microsoft Word, Anda harus menyimpan file dengan cara klik “File” lalu “Save As”Simpan file dengan format type “Web Page”Jika sudah, silakan buka aplikasi Excel Anda kemudian klik “file” lalu “open”Pilih file yang tadi sudah diubah menjadi versi web pageSelanjutnya, file akan terbuka di Excel dan Anda bisa mengedit ataupun menyimpannya menjadi bentuk file Excel.2.</fourth>
<fifth>Ubah PDF ke Excel langsungSelain cara di atas, juga ada cara lain untuk ubah file PDF menjadi Excel dengan langsung membukanya secara otomatis di program Microsoft Excel.Namun sebagai catatan, cara ini hanya bisa tersedia untuk pelanggan Office 365.Adapun cara ubah PDF ke Excel secara otomatis dikutip dari HowtoGeek yakni dengan memilih menu “Data” pada Excel.Selengkapnya cara ubah data PDF ke Excel secara otomatis yakni sebagai berikut:Buka aplikasi Excel AndaKlik “Data” lalu pada menu “Get Data” klik tombol panah naik turun dan klik “From File” selanjutnya klik “From PDF”Pilih dokumen PDF yang ingin Anda buka lalu klik “Import”Selanjutnya Anda akan dibawa pada kotak Navigator untuk memilih tabel yang Anda perlukan untuk dimasukkan ke dalam dokumen ExcelSelanjutnya pilih “Load to” dan klik “OK”Dengan cara ini Anda bisa membuka data PDF di Excel kemudian menyimpannya ke dalam bentuk Excel.</fifth>
